# Proyecto NOC

## dev

1. Clonar el archivo .env.template a .env
2. Configurar las variables de entorno
3. Ejecutar el comando

```
npm install
```

4. Ejecutar las bases de datos con el comando

```
docker compose up -d
```

5. Ejecutar

```
npx prisma migrate dev
```

6. Ejecutar

```
npm run dev
```
